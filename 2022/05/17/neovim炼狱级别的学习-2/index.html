neovim炼狱级别的学习(2)

Colemak 用户使用的 NeoVim 配置文件

安装此配置后你需要做的事

  • 安装 pynvim (使用 pip)
  • 安装 nodejs, 然后在终端输入 npm install -g neovim
  • 安装 nerd-fonts (尽管它是可选的，但是安装之后看上去十分地酷)
  • 安装hombrew然后在终端输入brew install neovim

    安装此配置后你可能想做的事

    首先

  • 执行 :checkhealth
  • 确保你安装了 Python
  • 查看 _machine_specific.vim 文件

    标签表

  • 安装 ctags 以获得类/函数/变量的三重支持

FZF

  • 安装 fzf

其它…

  • 安装 figlet 以输入 ASCII 艺术字
  • 安装 xclip 以获得系统剪切板访问支持 (仅 Linux 与 xorg 需要)

快捷键

最基本的键位

i : 切换至 插入 : 一种模式 (在原版 Vim 中与 i 键相同)

Q : 退出当前 Vim 窗口 (在原版 Vim 中与命令 :q 相同)

S : 保存当前文件 (在原版 Vim 中与命令 :w 相同)

改变了的光标移动方式

快捷键 行为
k 将光标向上移动一行
j 将光标向下移动一行
h 将光标向左移动一格
l 将光标向右移动一格
K 将光标向上移动五行
J 将光标向下移动五行
H 将光标移至当前行的第一个字符
L 将光标移至当前行的最后一个字符
Ctrl y 将视角向上移动五行而不移动光标
Ctrl e 将视角向下移动五行而不移动光标
e 将光标移至当前单词的末尾
W 将光标移至五个单词后的末尾
B 将光标移至五个单词前的开头
zz 将光标居中

在插入模式下的键位下的键位

快捷键 行为
Ctrl a 将光标移至当前行的末尾
Ctrl u 将光标所在的字母移至当前行的末尾

改变

快捷键 行为
u 撤销
< 反向缩进
> 缩进
SPACE SPACE 删除下一个 <++> 并进入插入模式

其它在普通模式下有用的键位

快捷键 行为
r 编译/运行当前文件
SPACE t t 将四个空格转换为制表符
SPACE o 最大窗口
Control t 在当前窗口下方新建一个终端窗口

增加了在可视模式下的命令

  • Y 复制选中文本至系统剪切板

窗口管理

通过分裂屏幕创造窗口

快捷键 行为
s k 新建一个分屏并把它放置在当前窗口的上面
s j 新建一个分屏并把它放置在当前窗口的下面
s h 新建一个分屏并把它放置在当前窗口的左边
s l 新建一个分屏并把它放置在当前窗口的右边
s v 将两个分屏垂直放置
s h 将两个分屏水平放置
s r v 将所有分屏垂直放置
s r h 将所有分屏水平放置

切换不同的窗口

快捷键 行为
SPACE + w 移至下一个窗口
SPACE + h 移至左边的窗口
SPACE + l 移至右边的窗口
SPACE + k 移至上面的窗口
SPACE + j 移至下面的窗口

为不同的窗口调整大小

用方向键更改当前窗口的大小

关闭窗口

  • Q 关闭当前窗口
  • SPACE q关闭当前窗口下面的窗口 (如果下面没有窗口，则当前窗口将关闭)

标签页管理

快捷键 行为
t k 新建一个标签页
t h 移至左侧标签页
t l 移至右侧标签页
t m h 将当前标签页向左移动一格
t m l 将当前标签页向右移动一格

自动补全

COC (自动补全)

快捷键 行为
Space y 打开剪切板历史

Ultisnips

快捷键 行为
Ctrl e 召唤一个代码片段
Ctrl n 在一个代码片段中前移光标
Ctrl e 在一个代码片段中后移光标

文件浏览

coc-explorer - 文件浏览器

快捷键 行为
tt 打开文件浏览器
? 查看帮助

FZF - 模糊文件查找器

快捷键 行为
leader f 模糊查找文件
Control j 向上移动一格
Control k 向下移动一格
leader w 模糊查找缓冲区
leader f 模糊查找文件内容

Undotree

快捷键 行为
Shift + m 打开撤回历史
T 显示修改时间
<tab> 移动光标
q 删除当前undotree

vim-multiple-cursors

快捷键 行为
Ctrl+n 选择下一个键 (多重光标)
Alt+k 选择所有键 (多重光标)
Ctrl+N 选择上一个键
Ctrl+s 跳过键
Esc 退出多重光标

vim-surround

想要添加包裹符号 (string -> "string"):

按下: ysiw':

想要修改包裹符号

按下: cs'":

按下: ds'":ds”删除分隔号

按下:yss+(:(是有空格的)

coc-translator

按下 ts 来 翻译光标所在的单词.

自定义代码片段补全

Markdown

快捷键 创建的文字
mb 粗体文字
mbi 斜粗体文字
mc 代码块
mp 图片
ml 链接
mh1 # H1
mh2 ## H2
mh3 ### H3
mh4 #### H4
[1-9][1-9]tb 4*4的表格

一些奇怪的东西

按 tx 然后输入你想要的文字

tx Hello<Enter>
